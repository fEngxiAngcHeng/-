<div class="wdgt" hide-btn="true">


Hides the widget body for the widget you place it on

<div class="wdgt" close-btn="true">


Fades out and closes the widget you place it on

<div class="wdgt" ajax-btn="true" ajax-path="file.html">


Loads the file listed at the "ajax-path" into the widget body and replaces the current content
